### v1.6.6 播放器注入策略重构

**核心改动：**

1. **iframe 使用 fixed 定位漂浮**：
   - 不再注入播放器容器，而是 `position: fixed` 定位在 body 上
   - 使用 `getBoundingClientRect()` 动态获取播放器位置
   - `z-index: 2147483647` 最大层级确保覆盖
   - 支持滚动、resize 时动态调整位置

2. **原视频彻底隐藏**：
   ```javascript
   el.muted = true;
   if (!el.paused) el.pause();
   el.style.opacity = '0';
   el.style.pointerEvents = 'none';
   ```

3. **守护进程双频策略**：
   - 前5秒：50ms 高频轮询（快速稳定注入）
   - 5秒后：250ms 低频轮询（持续监控）

4. **主动式换集检测**：
   ```javascript
   document.addEventListener('click', (event) => {
     const anchor = event.target.closest('a');
     if (anchor?.href?.includes('iqiyi.com/v_')) {
       // 自动重新解析
     }
   }, true);
   ```

5. **其他优化**：
   - 添加 `<base target="_self">` 确保链接在当前页打开
   - 使用可复用的 style 标签批量注入 CSS
   - 将用户指定的6个接口排在前面优先使用
