## 完成事项

### JavDB Manager 优化 (javdb-manager.user.js)

本次 session 完成以下优化（代码已 git commit）：

1. **统一持久化机制**：`loadConfig/saveConfig` 改用 `GM_getValue/GM_setValue`，配置通过 `DataCache.invalidate()` 清除缓存

2. **缩小 MutationObserver 范围**：只观察 `.movie-list` 容器，新增 `popstate` 事件监听 SPA 路由变化

3. **添加番号提取错误隔离**：`applyBlockEffect` 每项处理用 `safeExecute` 包装，单项异常不影响整页

4. **清理死代码**：
   - 删除 `throttle` 函数（未使用）
   - 删除 `PAGE_TYPE` 枚举（合并到 `LIST_TYPE`）
   - 删除 `debugScoreStructure` 函数及 UI 按钮
   - 删除 `CONFIG.DEBUG` 相关代码

5. **多域名支持**：
   - `@match` 改为 `https://javdb*.com/*` 匹配所有 javdb 域名
   - 添加 `getJavdbBaseUrl()` 函数动态获取当前域名
   - 跳转链接改用动态域名（`${baseUrl}/users/...`）
