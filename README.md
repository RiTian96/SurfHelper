# SurfHelper

> **从零开始改善你的上网体验**

个人自用备份仓库，包含各种实用工具和脚本。

## 📁 项目结构

```
SurfHelper/
├── tampermonkey-scripts/          # 油猴脚本文件夹
│   ├── coc-layout-helper.user.js  # COC阵型辅助脚本
│   ├── weibo-magnet-linker.user.js # 微博磁力链自动补全
│   └── video-parser.user.js       # 视频解析器
└── README.md                       # 项目说明文档
```

## 🛠️ 当前内容

### 油猴脚本
- **COC阵型辅助** - 绕过付费限制，一键复制阵型链接，支持大图预览和历史记录
- **微博磁力链自动补全** - 自动识别并补全微博中的磁力链接
- **视频解析器** - 多平台视频解析，集成多个解析接口
