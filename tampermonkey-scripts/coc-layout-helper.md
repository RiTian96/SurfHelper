# COC阵型复制助手

## 🔗 仓库导航
- **GitHub仓库**: [https://github.com/RiTian96/SurfHelper](https://github.com/RiTian96/SurfHelper)
- **Greasy Fork**: [https://greasyfork.org/zh-CN/users/332142-ritian96](https://greasyfork.org/zh-CN/users/332142-ritian96)

## ✨ 主要功能
- **无感提取** - 后台静默获取阵型链接，无需付费或登录
- **巨型预览** - 鼠标悬停显示高清大图，自适应屏幕尺寸
- **历史背包** - 左侧悬浮背包自动记录复制的阵型（最多50条）
- **智能避让** - 大图智能避开鼠标位置，不影响浏览
- **扫码直连** - 支持二维码扫码直接访问阵型

## 🚀 如何使用
1. **安装脚本** - 点击[安装链接](https://raw.githubusercontent.com/RiTian96/SurfHelper/main/tampermonkey-scripts/coc-layout-helper.user.js)，在油猴插件中安装
2. **获取阵型** - 访问[黑羽COC](https://coc.6oh.cn/)，点击"复制链接"按钮，链接自动复制到剪贴板
3. **预览大图** - 鼠标悬停在阵型卡片上查看高清大图
4. **查看历史** - 点击左侧背包图标 🎒 查看所有已复制的阵型历史
