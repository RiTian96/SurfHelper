# COC 阵型辅助

## 📋 脚本介绍

COC阵型辅助脚本是一款为 Clash of Clans (部落冲突) 阵型网站设计的增强工具，提供多项实用功能来改善用户体验。

## ✨ 主要功能

### 🎯 核心功能
- **绕过付费限制** - 后台无感提取阵型链接，无需付费或登录
- **一键复制** - 快速复制阵型链接，方便分享和保存
- **智能大图预览** - 鼠标悬停显示高清巨型大图，自适应尺寸，智能避让鼠标
- **侧边背包** - 左侧悬浮背包记录历史阵型，支持二维码扫码直连

### 🛠️ 辅助功能
- **历史记录管理** - 自动保存浏览过的阵型，最多保存50条
- **调试模式** - 可开启调试日志，便于问题排查
- **响应式设计** - 适配各种屏幕尺寸

## 🚀 安装使用

### 安装步骤
1. 确保已安装 [Tampermonkey](https://www.tampermonkey.net/) 油猴插件
2. 点击 [🔧 安装脚本](coc-layout-helper.user.js)
3. 在弹出的页面中点击"安装"

### 使用方法
1. 访问支持的COC阵型网站
2. 脚本会自动运行，无需额外操作
3. 享受各项增强功能

## 🌐 支持的网站

- `coc.6oh.cn`
- `*.6oh.cn` (所有子域名)

## ⚙️ 配置选项

脚本提供以下可配置选项：

```javascript
const CONFIG = {
    bagMaxItems: 50,    // 历史记录最大保存条数
    debug: false        // 是否开启调试日志
};
```

## 🔧 技术特性

- **无依赖运行** - 不依赖外部库，轻量高效
- **安全可靠** - 仅在前端运行，不收集用户数据
- **性能优化** - 智能DOM操作，不影响页面性能
- **兼容性强** - 支持现代浏览器
