# 微博磁链补全助手

## 🔗 仓库导航
- **GitHub仓库**: [https://github.com/RiTian96/SurfHelper](https://github.com/RiTian96/SurfHelper)
- **Greasy Fork**: [https://greasyfork.org/zh-CN/users/332142-ritian96](https://greasyfork.org/zh-CN/users/332142-ritian96)

## ✨ 主要功能
- **智能识别** - 自动识别页面中的40位磁力哈希值
- **自动补全** - 将哈希值自动转换为完整的磁力链接
- **一键点击** - 补全后的链接可直接点击下载
- **无感运行** - 后台自动处理，无需手动操作
- **安全过滤** - 避免在链接、代码等元素中误匹配

## 🚀 如何使用
1. **安装脚本** - 点击[安装链接](https://raw.githubusercontent.com/RiTian96/SurfHelper/main/tampermonkey-scripts/weibo-magnet-linker.user.js)，在油猴插件中安装
2. **访问微博** - 打开微博网站（weibo.com、s.weibo.com、d.weibo.com）
3. **自动补全** - 脚本自动运行，页面中的磁力哈希值会自动转换为可点击链接
4. **点击下载** - 直接点击链接即可调用下载工具
