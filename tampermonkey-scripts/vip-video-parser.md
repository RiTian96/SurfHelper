# VIP视频解析器

## 🔗 仓库导航
- **GitHub仓库**: [https://github.com/RiTian96/SurfHelper](https://github.com/RiTian96/SurfHelper)
- **Greasy Fork**: [https://greasyfork.org/zh-CN/users/332142-ritian96](https://greasyfork.org/zh-CN/users/332142-ritian96)

## ✨ 主要功能
- **多平台支持** - 支持腾讯视频、爱奇艺、优酷、B站、芒果TV
- **15个解析接口** - 内置多个高质量解析接口，支持切换和评分
- **智能过滤** - B站仅解析付费内容，不影响普通视频观看
- **剧集自动切换** - 检测剧集切换自动重新解析
- **智能错误处理** - 详细错误分类和自动重试机制

## 🚀 如何使用
1. **安装脚本** - 点击[安装链接](https://raw.githubusercontent.com/RiTian96/SurfHelper/main/tampermonkey-scripts/vip-video-parser.user.js)，在油猴插件中安装
2. **访问视频网站** - 打开支持的视频网站（腾讯、爱奇艺、优酷、B站、芒果TV）
3. **点击解析按钮** - 在视频播放页面点击"解析视频"按钮
4. **选择解析接口** - 在弹出面板中选择解析接口，享受解析后的视频内容
