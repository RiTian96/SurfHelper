## ✨ 主要功能
- **看过/想看屏蔽** - 自动识别并降低看过/想看影片的透明度，避免重复浏览
- **智能评分分级** - 基于评价人数的动态阈值，人数越多判断越精准；低分变暗，高分绿色边框高亮
- **批量导入** - 支持从看过/想看页面批量导入影片列表，智能翻页
- **数据备份** - 支持导出/导入番号数据到本地JSON文件，合并或覆盖模式可选
- **搜索精确高亮** - **[2.0新功能]** 智能识别搜索词，自动忽略大小写、空格及横杠；支持纯数字匹配 FC2 番号；匹配项以“金边呼吸灯”特效突出显示
- **详情页联动** - **[2.0新功能]** 自动识别页面“看过/想看”标签；点击按钮同步导入；点击删除按钮同步从本地移除
- **大图预览** - 鼠标悬停封面显示高清大图，智能避让自适应屏幕，支持开关控制
- **页面区分屏蔽** - 看过页面不屏蔽想看影片，想看页面不屏蔽看过影片

## 🚀 如何使用
1. **安装脚本** - 点击[安装链接](https://raw.githubusercontent.com/RiTian96/SurfHelper/main/tampermonkey-scripts/javdb-manager.user.js)，在油猴插件中安装
2. **搜索影片** - 在搜索框输入番号进行检索，精确匹配的结果将获得金色流光边框
3. **导入影片** - 访问JavDB的「看过影片」或「想看影片」页面，点击右上角管理器图标，选择对应导入按钮
4. **管理影片** - 在任意页面点击管理器图标，切换到"管理"标签页，搜索并添加/删除影片
5. **数据备份** - 在导入页面点击"导出数据"备份番号，点击"导入数据"恢复备份
6. **自定义设置** - 在管理面板中开启/关闭各项功能，设置自动保存
