## ✨ 主要功能

JavDB影片管理器专注于帮助你在 JavDB 网站上高效管理观影记录。基于评价人数的智能评分系统自动筛选优质影片，配合「看过/想看」自动屏蔽，让浏览体验更专注。

**核心功能：**
- **🛡️ 影片屏蔽** — 自动屏蔽已标记「看过/想看」的影片，降低透明度避免重复浏览，支持分区隔离
- **📊 智能评分** — 基于评价人数的动态阈值，高分影片绿边高亮，低分影片自动隐身
- **🔍 搜索精确高亮** — 搜索结果精确匹配项展示黄金特效，支持 FC2 纯数字格式
- **🔄 详情页同步** — 详情页「看过/想看」按钮点击后自动导入/删除，支持冲突数据自动清洗
- **👁️ 悬浮大图** — 鼠标悬停封面即显高清大图，智能避让光标，自适应屏幕位置
- **📥 数据管理** — 一键爬取个人主页列表，支持合并/覆盖双模式导入，JSON 数据完整备份与恢复
- **🌍 全码支持** — 完美支持日产、欧美及 FC2 等各类番号格式
- **🎨 玻璃拟态 UI** — Glassmorphism 界面设计，适配深色模式

## 🚀 如何使用

1. **安装脚本** - 点击 [安装链接](https://raw.githubusercontent.com/RiTian96/SurfHelper/main/tampermonkey-scripts/javdb-manager.user.js) 在油猴插件中安装
2. **搜索影片** - 在搜索框输入番号，精确匹配的结果将获得**金色流光边框**
3. **导入数据** - 访问 JavDB「看过/想看」页面，点击右上角浮窗，选择**导入**功能
4. **管理数据** - 点击浮窗切换至**管理**页，可查询状态、手动添加/删除或备份数据
5. **功能开关** - 在管理面板中可实时开启/关闭屏蔽、评分、大图预览等功能
